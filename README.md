# Overview

Nine Virtual Technologies website built using the following:

- Framework: NextJS
- CMS: Contentful
- Deployment: Github & Vercel

# Getting Started

1. Install Dependencies: `npm install` or `yarn`
2. Run the Development Server: `npm run dev` or `yarn dev`
